将 SRT 字幕文件完整转换为 Markdown 笔记，保留全部原文，添加标点、段落格式和截图占位符。

## 第一步：读取并解析 SRT 文件

读取用户提供的 `.srt` 字幕文件，提取所有文本内容和对应的时间戳。

## 第二步：转换为 Markdown 格式

按以下规则转换：

### 内容规则

- **完整保留**：禁止任何删减、总结或省略，必须逐字保留 SRT 所有文字。
- **语言**：使用中文书写，保留必要的英文专有名词。

### 段落结构

- 仅使用一个层级的段落标题（`## 标题`）。
- 第一段为引言，不使用标题。
- 整篇划分为 2–3 个段落。
- 为每句话添加合适的标点，适当划分自然段。

### 截图占位符

当某句符合以下任一条件时，在句末添加截图标记：

- 代码讲解
- UI 交互操作
- 包含”这么、这里、这儿”等视觉指代词
- 提及网址、链接、地址（包括 GitHub、API endpoint 等）
- 任何借助视觉材料更易理解的内容

格式必须严格为 `Screenshot-[hh:mm:ss]`，时间点取该句的末尾时间戳。仅在真正有助于理解时插入，不可滥用。

**示例**：

输入 SRT：
```
00:02:08,600 --> 00:02:09,800
安装这个工具
```

输出 Markdown：
```
安装这个工具。Screenshot-[00:02:09]
```

## 第三步：保存输出文件

将生成的 Markdown 保存到项目根目录，文件名基于视频主题命名，如 `项目根目录/<视频主题>.md`。

## 第四步：验证输出

保存后检查：

- Markdown 文件是否包含 SRT 中的所有文字（无遗漏）
- 截图占位符格式是否正确（`Screenshot-[hh:mm:ss]`）
- 段落结构是否为 2–3 个 `##` 标题段落

## 第五步：生成视频截图

验证通过后，运行以下命令从视频中提取截图：

```bash
python scripts/screenshot.py
```

不需要传递任何参数，脚本会自动读取 Markdown 中的截图占位符并生成对应截图。
